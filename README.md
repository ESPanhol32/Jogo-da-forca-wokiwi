# Jogo-da-forca-wokiwi

Esse jogo possui 45 possibilidades de palavras diferentes
O jogador deve tomar cuidado para acertar as letras e descobrir a palavra secreta

As letras vão de "A" a "Z" e são selecionadas através dos botões que funcionam assim:
1- próxima letra
2- letra anterior
3- confirma

Caso o jogador confirme uma letra que não pertence á palavra, ele perde uma vida, o jogador tem um total de 6 vidas.
Caso o jogador acerte todas as letras antes de perder todas as vidas, ele ganha!
